HomeworkTracker

Beta · Criterion C · D.Yedil · S.Khamza

01. What is it?

App

Smart homework tracker for students 10–17 with gamification and parental control.

Problem

Students forget assignments and rush at the last minute, causing stress and lower grades.

Unique

Free, no install, Supabase cloud on any device, dark mode.

Audience

Students 10–17 who want a simple tool on phone and computer.

02. Tech Stack

Features

  • Supabase Auth + Database
  • Real-time cloud sync
  • Priority system
  • Search & filters
  • 15+ Achievements
  • Parental Control
  • Dark Mode

Technical

  • HTML5 / CSS3 / Vanilla JS
  • Supabase — Auth + Postgres
  • Real-time subscriptions
  • No frameworks
